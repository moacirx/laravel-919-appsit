LARAVEL 9 adminLTE

  1. Instale laravel ultima versão
  2. acerte o banco de dados no .env
  3. composer require jeroennoten/laravel-adminlte
  4. php artisan adminlte:install
  5. composer require laravel/ui
  6. php artisan ui vue --auth
  7. npm install
  8. para corrigir o erro do form plugins: [require('@tailwindcss/forms'), require('@tailwindcss/typography')], que esta no arquivo tailwind.config.js use esse passo abaixo

    npm install autoprefixer@10.4.5 --save-exact

  9. npm run dev

    Erro : Error: Cannot find module 'webpack/lib/rules/DescriptionDataMatcherRulePlugin'

    solução : npm update vue-loader e depois npm i vue-loader

  10. php artisan migrate
  11. composer require lucascudo/laravel-pt-br-localization --dev
  12. php artisan vendor:publish --tag=laravel-pt-br-localization
  13. alterar a localidade em config/app.php: 'locale' => 'pt-BR' e 'timezone' => 'America/Sao_Paulo'
  14. php artisan adminlte:status
  15. php artisan adminlte:install --only=auth_views
  16. composer update jeroennoten/laravel-adminlte
  17. php artisan adminlte:update
  18. php artisan adminlte:install --only=main_views --force
  19. php artisan adminlte:install --only=config
  20. php artisan adminlte:install --only=translations --force
    21. Faker -> alterar em config.app = 'faker_locale' => 'pt_BR',
